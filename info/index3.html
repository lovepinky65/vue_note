v-for

v-for 基本寫法 v-for="item in list" (v-for="自訂變數 in 陣列名")

v-for 索引寫法(從0開始) v-for="(item,index) in list" (v-for="(自訂變數,index) in 陣列名")

範例

基本寫法
  • {{ item.name }} 的年齡是 {{ item.age }}


索引寫法(從0開始)
  • {{ index+1 }} - {{ item.name }} 的年齡是 {{ item.age }}

v-if

v-if 寫法 v-if="item.age < 50"

範例

判斷
  • {{ index+1 }} - {{ item.name }} 的年齡是 {{ item.age }}
